This directory contains comprehensive research reports generated by Claude Deep Research workflow.

## ğŸ“š About Claude Deep Research

The Claude Deep Research workflow is an automated system that uses Claude AI with web search capabilities to conduct thorough research on any topic and generate detailed markdown reports.

### Features

- **Web Search Integration**: Uses WebSearch tool to find current, authoritative information
- **Comprehensive Analysis**: Synthesizes information from multiple sources
- **Professional Reports**: Generates well-structured markdown documents
- **Automated Workflow**: Triggered via GitHub Issues/PRs or manual dispatch
- **Version Control**: Creates PRs with research findings for review

### Method 1: Comment Trigger (Issues/PRs)

Comment on any issue or pull request with:

```
@claude-research [Your Research Topic]
```

**Examples:**
```
@claude-research AI safety trends in 2024
@claude-research Latest developments in quantum computing
@claude-research Best practices for microservices architecture
```

### Method 2: Manual Workflow Dispatch

1. Go to **Actions** tab in GitHub
2. Select **Claude Deep Research** workflow
3. Click **Run workflow**
4. Enter:
   - **Research Topic**: The topic you want researched
   - **Output Filename**: Custom filename (optional)
5. Click **Run workflow**

## ğŸ“‹ What Gets Generated

Claude will:

1. **Perform Multiple Web Searches** (3-5+ different queries)
2. **Gather Information** from authoritative sources
3. **Analyze and Synthesize** findings
4. **Create Markdown Report** with:
   - Executive Summary
   - Key Findings
   - Detailed Analysis
   - Current State & Future Outlook
   - Conclusions
   - Sources/Citations
5. **Commit Report** to `research/` directory
6. **Create Pull Request** for review
7. **Comment on Original Issue** with results
